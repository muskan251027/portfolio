EQUIST - Equitable Impact Sensitive Tool (UNICEF) UNICEF has designed and developed EQUIST as an analytical platform with free online access to facilitate decision makers' development of equitable strategies in order to improve health and nutrition of the women and children in most vulnerable situations. SAFER ROADS (City Intervention In India) An industry-wide, cooperative project called Safer Roads for Gurugram (SRFG) is being led by AB InBev (Anheuser-Busch InBev) as the organising body. The initiative's main goal is to make Gurugram's roadways safer through data-driven initiatives and constructive behavioural change. It enables dedicated groups to support the development, implementation, monitoring, and evaluation of various road safety projects through a public-private collaboration. The initiative will adopt a dual-pronged strategy, concentrating on both bolstering the current procedures and systems and assisting in the creation & implementation of distinctive and new solutions.
UN INFO (UNDP) It is an online planning, monitoring and reporting system that digitizes each UNDAF/UNSDCF and the Joint work plans that go with them. The UN development system's efforts to increase coherence, openness, and accountability are reflected in this, which will allow it to better meet the needs and priorities of Member States. UN INFO, which was first designed and created at the national level, gives an overview of how the UN development system in each nation is allocating resources to national priorities, thematic sectors (such as those that cut across gender, human rights, and humanitarian criteria), and the SDGs.

MovieSearch This is a platform based on Node.js which allows the user to search movies based on the list of selected genres and entered title. The user is given a list of genres and they can select multiple in order to search the movies. Also, the user will be required to give some title in order to retrieve a list of movies. There are two public apis that have been used in this project - OTT DETAILS API(to get the list of movies) and GIPHY API(to get stickers for user interface for the list of genres)
Say Cheese A game made in HTML, CSS and JS to make the rat escape all obstacles in its way to reach cheese. There is a set of rules that need to be followed in this game in order to win -
  • The player (rat) needs to reach cheese in order to win the game.
  • Use arrow keys to move the player.
  • The moving bells are obstacles and the player has to avoid them to prevent losing a life.
  • The player has got three lives to win the game.
  • Every time the player touches the bell, it loses a life.
FRIDGETERIA (Your own custom refrigerator) This is basically a customized refrigerator based on JQuery, JS, HTML and CSS which gives recipe suggestions according to the ingredients added to refrigerator. The list of features include - You need to submit user details before entering, like your name and the name you want to call your fridge with (just for fun). After successful validation of your form, click continue and it will take you to the first page. First page has all the ingredients list that user can select from to add items in fridge. User can add or remove items according to them. Clicking on Proceed will take you to the second page, which basically displays your final ingredient list. On second page, clicking edit button will take you back to ingredient selection part and suggest recipies will take you to next slide - suggestions. Final slide, it will display the suggestions according to your selected ingrdients. Press Back on Last slide takes the user back to last page. Switching between radio buttons list items help user to categroize recipes between food or beverage or all.
